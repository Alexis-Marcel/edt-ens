# EDT ENS SAPHIRE

Interactive timetable for the SAPHIRE program at ENS Paris-Saclay.

- Automatic parsing of the Google Sheets schedule (CSV export)
- Calendar display with [FullCalendar](https://fullcalendar.io/)
- Filtering by class (GC, GM, GE)
- Group and multi-room detection (English groups, project rooms, TP groups)

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
